# 🍵 Bae Rituals – Premium Matcha Experience

A premium Gen Z-focused DTC (Direct-to-Consumer) ecommerce website designed for **Bae Rituals**, a modern ceremonial matcha brand.

The project combines immersive storytelling, smooth animations, premium UI design, and AI-generated creative assets to create a high-end shopping experience inspired by brands like Apple, Rhode, Aesop, and Chamberlain Coffee.

## 🌐 Live Demo

🔗 https://matcha-website-sigma.vercel.app/

- 🎨 Premium modern UI/UX
